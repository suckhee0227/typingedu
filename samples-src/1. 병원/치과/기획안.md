# 기획안 — 치과: 발치·임플란트 후 관리 스마트 안내

## 콘텐츠 개요
- **콘텐츠명**: 발치·임플란트 후 관리 스마트 안내
- **대상**: 치과 내원 환자/보호자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 2~3분
- **콘텐츠 의도**: 핵심 안내 → 체크리스트 → 퀴즈로 이어지는 단계형 구성으로,
  환자가 스스로 확인·체크·참여하며 핵심 관리 포인트를 기억하도록 돕는다.
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 콘텐츠 제목과 간단한 안내 문구 표시 ("치료 후 관리 포인트를 쉽고 빠르게 확인해보세요!")
- 이용 안내 3종(단계별 가이드 / 체크리스트 / 퀴즈) 노출
- '시작하기' 버튼 → 핵심 안내 카드로 이동
- 분위기: 부드럽고 깨끗한 의료 톤(테마색 #18B0C4)

### 2단계) 핵심 안내 카드 (5장 스와이프)
  1. 🩹  — 거즈는 안내받은 시간 동안 물고 있어요.
  2. 🥤  — 빨대 사용은 피해주세요.
  3. 🚭  — 음주와 흡연은 최소 3일간 금지예요.
  4. 🥣  — 부드러운 음식으로 식사하세요.
  5. 🦷  — 통증·출혈이 심하면 병원에 연락하세요.
- 카드별 아이콘·도트 인디케이터, 이전/다음 네비게이션 제공

### 3단계) 관리 체크리스트
- 환자가 직접 체크하는 체크리스트(모두 체크해야 다음 단계 이동)
  - 🩹 거즈를 안내 시간 동안 물고 있었어요
  - 🥤 빨대 사용을 하지 않았어요
  - 🚭 음주와 흡연을 피하고 있어요
  - 🥣 부드러운 음식을 섭취하고 있어요
  - 🦷 통증이나 출혈이 심하면 병원에 연락할 거예요

### 4단계) 확인 퀴즈 (OX 3문항)
  1. (정답 X) 발치 후 빨대를 사용하면 출혈이 멈추기 쉬워진다?
     → 빨대 사용은 음압을 만들 수 있어 출혈이 다시 생길 수 있으므로 피해야 합니다.
  2. (정답 O) 발치 후에는 음주와 흡연을 피하는 것이 좋다?
     → 음주·흡연은 회복을 늦추므로 최소 3일간 피해주세요.
  3. (정답 X) 수술 당일에는 단단한 음식을 마음껏 먹어도 된다?
     → 상처를 자극할 수 있어요. 부드러운 음식으로 드세요.

### 5단계) 완료 화면
- "퀴즈 완료!" + 점수 표시(예: 3/3)
- 병원 문의 안내: OO 치과 / 02-123-4567 / 평일 09:00~18:00
- '처음으로 돌아가기' 버튼으로 복습 가능

## 디자인 가이드
- 컬러: 테마색 #18B0C4 계열 + 화이트 배경, 부드러운 의료 톤
- 폰트: 가독성 높은 한글 산세리프
- 캐릭터: 친근한 물방울 캐릭터(테마색 톤)
- 16:9 가로 화면 최적화, 손가락으로 누르기 쉬운 큰 버튼

## 기술 구성
- 공용 데이터 기반 엔진(app.ts) + 병원별 data.json
- 단계 전환은 화면 내에서 자연스럽게 처리, 별도 외부 라이브러리 의존 없음
