# 기획안 — 정형외과: 깁스·수술 후 회복관리 스마트 안내

## 콘텐츠 개요
- **콘텐츠명**: 깁스·수술 후 회복관리 스마트 안내
- **대상**: 정형외과 내원 환자/보호자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 2~3분
- **콘텐츠 의도**: 핵심 안내 → 체크리스트 → 퀴즈로 이어지는 단계형 구성으로,
  환자가 스스로 확인·체크·참여하며 핵심 관리 포인트를 기억하도록 돕는다.
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 콘텐츠 제목과 간단한 안내 문구 표시 ("회복을 돕는 관리 포인트를 쉽고 빠르게 확인해보세요!")
- 이용 안내 3종(단계별 가이드 / 체크리스트 / 퀴즈) 노출
- '시작하기' 버튼 → 핵심 안내 카드로 이동
- 분위기: 부드럽고 깨끗한 의료 톤(테마색 #3B7DD8)

### 2단계) 핵심 안내 카드 (5장 스와이프)
  1. 🦵  — 깁스가 젖지 않도록 주의하세요.
  2. 🦶  — 부기와 통증이 심해지면 병원에 문의하세요.
  3. 🩼  — 목발은 안내받은 방법대로 사용하세요.
  4. 🛋️  — 무리한 활동이나 운동은 피해주세요.
  5. 📅  — 정해진 진료 일정은 꼭 지켜주세요.
- 카드별 아이콘·도트 인디케이터, 이전/다음 네비게이션 제공

### 3단계) 관리 체크리스트
- 환자가 직접 체크하는 체크리스트(모두 체크해야 다음 단계 이동)
  - 🩹 통증이 심해지거나 지속되면 병원에 연락하기
  - 🦴 깁스·보호대가 불편하거나 손상되면 병원에 알리기
  - 💊 처방받은 약을 정해진 시간에 복용하기
  - ☝️ 다리를 높여 부기를 줄이도록 노력하기
  - 🚶 무리한 활동이나 운동을 하지 않기

### 4단계) 확인 퀴즈 (OX 3문항)
  1. (정답 X) 깁스한 부위의 발가락이 창백해지거나 차가워져도 그냥 지켜보면 된다?
     → 발가락이 창백하거나 차가워지는 것은 혈액순환 문제의 신호일 수 있어요. 즉시 병원에 연락하세요.
  2. (정답 O) 다친 다리를 높이 올리면 부기 완화에 도움이 된다?
     → 다친 부위를 심장보다 높게 두면 부기와 통증 완화에 도움이 됩니다.
  3. (정답 X) 목발은 편한 대로 아무렇게나 사용해도 된다?
     → 잘못 사용하면 다칠 수 있어요. 안내받은 방법대로 사용하세요.

### 5단계) 완료 화면
- "퀴즈 완료!" + 점수 표시(예: 3/3)
- 병원 문의 안내: OO 정형외과 / 02-123-4567 / 평일 09:00~18:00
- '처음으로 돌아가기' 버튼으로 복습 가능

## 기술 구성

- 단계 전환은 화면 내에서 자연스럽게 처리, 별도 외부 라이브러리 의존 없음
