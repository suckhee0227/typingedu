# 기획안 — 내과: 건강검진 전날 체크리스트 스마트 안내

## 콘텐츠 개요
- **콘텐츠명**: 건강검진 전날 체크리스트 스마트 안내
- **대상**: 내과 내원 환자/보호자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 2~3분
- **콘텐츠 의도**: 핵심 안내 → 체크리스트 → 퀴즈로 이어지는 단계형 구성으로,
  환자가 스스로 확인·체크·참여하며 핵심 관리 포인트를 기억하도록 돕는다.
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 콘텐츠 제목과 간단한 안내 문구 표시 ("검진 전날 준비사항을 쉽고 빠르게 확인해보세요!")
- 이용 안내 3종(단계별 가이드 / 체크리스트 / 퀴즈) 노출
- '시작하기' 버튼 → 핵심 안내 카드로 이동
- 분위기: 부드럽고 깨끗한 의료 톤(테마색 #4CAF6E)

### 2단계) 핵심 안내 카드 (5장 스와이프)
  1. 🍽️  — 검진 전날 저녁은 9시 이전까지 가볍게 드세요.
  2. 🥤  — 검진 8시간 전부터 금식하세요. (물은 소량 가능)
  3. 🚫  — 음주나 과식은 피해주세요.
  4. 💊  — 복용 중인 약이 있다면 의사와 상의하세요.
  5. 😴  — 충분한 수면을 취하고 편안한 상태를 유지하세요.
- 카드별 아이콘·도트 인디케이터, 이전/다음 네비게이션 제공

### 3단계) 관리 체크리스트
- 환자가 직접 체크하는 체크리스트(모두 체크해야 다음 단계 이동)
  - 🍽️ 저녁 식사는 오후 9시 이전에 가볍게 먹기
  - 🥤 검사 8시간 전부터 금식하기 (물은 소량만)
  - 🚫 음주나 과식을 하지 않기
  - 💊 복용 중인 약이 있다면 의료진과 상의하기
  - 😴 충분한 수면을 취하고 편안한 상태 유지하기

### 4단계) 확인 퀴즈 (OX 3문항)
  1. (정답 X) 검진 전날 저녁 식사는 자정까지 자유롭게 먹어도 된다?
     → 검진 전날 저녁은 9시 이전까지 가볍게 드세요. 너무 늦은 식사는 위장에 영향을 줄 수 있습니다.
  2. (정답 O) 금식 중에도 소량의 물은 마실 수 있다?
     → 물은 소량 마실 수 있어요. 단, 음료·우유 등은 피하세요.
  3. (정답 X) 매일 복용하는 약은 검진 전 무조건 끊어야 한다?
     → 약에 따라 다르므로 반드시 의료진과 상의해 결정하세요.

### 5단계) 완료 화면
- "퀴즈 완료!" + 점수 표시(예: 3/3)
- 병원 문의 안내: OO 내과 / 02-123-4567 / 평일 09:00~18:00
- '처음으로 돌아가기' 버튼으로 복습 가능

## 디자인 가이드
- 컬러: 테마색 #4CAF6E 계열 + 화이트 배경, 부드러운 의료 톤
- 폰트: 가독성 높은 한글 산세리프
- 캐릭터: 친근한 물방울 캐릭터(테마색 톤)
- 16:9 가로 화면 최적화, 손가락으로 누르기 쉬운 큰 버튼

## 기술 구성
- 공용 데이터 기반 엔진(app.ts) + 병원별 data.json
- 단계 전환은 화면 내에서 자연스럽게 처리, 별도 외부 라이브러리 의존 없음
