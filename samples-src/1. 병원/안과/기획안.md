# 기획안 — 안과: 안약 사용법·수술 후 주의사항 스마트 안내

## 콘텐츠 개요
- **콘텐츠명**: 안약 사용법·수술 후 주의사항 스마트 안내
- **대상**: 안과 내원 환자/보호자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 2~3분
- **콘텐츠 의도**: 핵심 안내 → 체크리스트 → 퀴즈로 이어지는 단계형 구성으로,
  환자가 스스로 확인·체크·참여하며 핵심 관리 포인트를 기억하도록 돕는다.
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 콘텐츠 제목과 간단한 안내 문구 표시 ("안약 사용법과 주의사항을 쉽고 빠르게 확인해보세요!")
- 이용 안내 3종(단계별 가이드 / 체크리스트 / 퀴즈) 노출
- '시작하기' 버튼 → 핵심 안내 카드로 이동
- 분위기: 부드럽고 깨끗한 의료 톤(테마색 #2FA98E)

### 2단계) 핵심 안내 카드 (5장 스와이프)
  1. 💧  — 안약은 처방된 시간과 횟수에 맞게 사용해요.
  2. 🧼  — 안약 넣기 전 손을 깨끗이 씻어요.
  3. 🙈  — 눈을 비비거나 만지지 않아요.
  4. 🕶️  — 외출 시 선글라스 착용, 자외선 차단을 해요.
  5. 🏥  — 통증, 시력 저하는 병원에 연락해요.
- 카드별 아이콘·도트 인디케이터, 이전/다음 네비게이션 제공

### 3단계) 관리 체크리스트
- 환자가 직접 체크하는 체크리스트(모두 체크해야 다음 단계 이동)
  - 💧 안약을 처방된 시간과 횟수에 맞게 사용하기
  - 🧼 안약 사용 전 손을 깨끗이 씻기
  - 🙈 눈을 비비거나 만지지 않기
  - 🕶️ 외출 시 선글라스 착용과 자외선 차단하기
  - 🏥 통증·시력 저하 등 증상이 있으면 병원에 연락하기

### 4단계) 확인 퀴즈 (OX 3문항)
  1. (정답 X) 안약을 넣은 직후 눈을 감지 않고 바로 깜빡거리는 것이 좋다?
     → 안약을 넣은 후에는 1~2분 눈을 감고 눈 안쪽을 살짝 눌러주면 흡수가 잘 됩니다.
  2. (정답 O) 안약을 넣기 전에는 손을 깨끗이 씻어야 한다?
     → 손의 세균이 눈에 들어가면 감염될 수 있어 꼭 씻어야 합니다.
  3. (정답 X) 수술 후 눈이 가려우면 비벼도 괜찮다?
     → 비비면 상처가 생길 수 있어요. 눈을 비비거나 만지지 마세요.

### 5단계) 완료 화면
- "퀴즈 완료!" + 점수 표시(예: 3/3)
- 병원 문의 안내: OO 안과 / 02-123-4567 / 평일 09:00~18:00
- '처음으로 돌아가기' 버튼으로 복습 가능

## 디자인 가이드

- 폰트: 가독성 높은 한글 산세리프
- 캐릭터: 친근한 물방울 캐릭터(테마색 톤)
- 16:9 가로 화면 최적화, 손가락으로 누르기 쉬운 큰 버튼

## 기술 구성
- 공용 데이터 기반 엔진(app.ts) + 병원별 data.json
- 단계 전환은 화면 내에서 자연스럽게 처리, 별도 외부 라이브러리 의존 없음
