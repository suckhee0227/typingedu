# 기획안 — 소아과: 아이 열날 때 대처 스마트 안내

## 콘텐츠 개요
- **콘텐츠명**: 아이 열날 때 대처 스마트 안내
- **대상**: 소아과 내원 환자/보호자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 2~3분
- **콘텐츠 의도**: 핵심 안내 → 체크리스트 → 퀴즈로 이어지는 단계형 구성으로,
  환자가 스스로 확인·체크·참여하며 핵심 관리 포인트를 기억하도록 돕는다.
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 콘텐츠 제목과 간단한 안내 문구 표시 ("아이가 열날 때 대처법을 쉽고 빠르게 확인해보세요!")
- 이용 안내 3종(단계별 가이드 / 체크리스트 / 퀴즈) 노출
- '시작하기' 버튼 → 핵심 안내 카드로 이동
- 분위기: 부드럽고 깨끗한 의료 톤(테마색 #5AB0E8)

### 2단계) 핵심 안내 카드 (5장 스와이프)
  1. 🌡️  — 38℃ 이상의 열이 있어요
  2. 🙂  — 열은 있지만 잘 먹고 놀아요
  3. 🥶  — 오한이 나고 몸이 떨려요
  4. 🤢  — 구토나 설사를 하고 있어요
  5. 😵  — 축 처지고 깨우기 힘들어요
- 카드별 아이콘·도트 인디케이터, 이전/다음 네비게이션 제공

### 3단계) 관리 체크리스트
- 환자가 직접 체크하는 체크리스트(모두 체크해야 다음 단계 이동)
  - 💊 해열제는 의사 지시 용량과 간격을 지키기
  - 💧 수분을 자주 섭취하게 해주기
  - 🌡️ 실내 온도를 적절하게 유지하기
  - 👀 아이 상태를 계속 관찰하기
  - 🏥 증상이 심해지면 병원에 연락하기

### 4단계) 확인 퀴즈 (OX 3문항)
  1. (정답 X) 아이가 38도 이상의 열이 있을 때는 무조건 해열제를 먹이는 것이 좋다?
     → 아이의 상태를 먼저 확인하고 필요할 때 해열제를 사용하는 것이 좋아요.
  2. (정답 O) 열이 날 때는 수분을 자주 섭취하게 하는 것이 좋다?
     → 열이 나면 수분이 부족해지기 쉬워요. 물·수분을 자주 먹여주세요.
  3. (정답 X) 아이가 축 처지고 깨우기 힘들면 집에서 지켜보면 된다?
     → 위험한 신호일 수 있어요. 즉시 병원에 가야 합니다.

### 5단계) 완료 화면
- "퀴즈 완료!" + 점수 표시(예: 3/3)
- 병원 문의 안내: OO 소아과 / 02-123-4567 / 평일 09:00~18:00
- '처음으로 돌아가기' 버튼으로 복습 가능

## 디자인 가이드

- 캐릭터: 친근한 물방울 캐릭터(테마색 톤)

## 기술 구성
- 공용 데이터 기반 엔진(app.ts) + 병원별 data.json
- 단계 전환은 화면 내에서 자연스럽게 처리, 별도 외부 라이브러리 의존 없음
