# 기획안 — 유아·초등 · 놀이학습: 그림 보고 말하기 스마트 학습

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 그림 보고 말하기 스마트 학습
- **과목/대상**: 유아·초등 · 놀이학습
- **대상 학년/연령**: 5세 · 6세 · 7세 · 초1 · 초2
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 2~3분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(🧸 유아·초등) + 자 진도바(✏️ 마커)
- 테마색 #F76707 + 포인트 #4DABF7

### 2단계) 그림 낱말 카드 (6장, 플래시카드)
  1. 🐘 코끼리 — 코가 길고 몸집이 큰 동물이에요.
  2. 🍎 사과 — 빨갛고 동그란 맛있는 과일이에요.
  3. 🚗 자동차 — 바퀴로 달리는 탈것이에요.
  4. ⭐ 별 — 밤하늘에서 반짝반짝 빛나요.
  5. 🌸 꽃 — 예쁜 색으로 피어나는 식물이에요.
  6. 🐶 강아지 — 멍멍 짖는 귀여운 동물이에요.

### 3단계) 오늘 한 일
  - 🐘 동물 이름을 말했어요
  - 🍎 과일 이름을 말했어요
  - 🚗 탈것 이름을 말했어요
  - 🌸 자연 낱말을 말했어요
  - 🎨 색깔을 말했어요
- 우측: 💡 TIP + '퀴즈하러 가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 코끼리는 코가 길까요?
     → 맞아요! 코끼리는 코가 아주 길어요.
  2. (정답 X) 사과는 파란색 일까요?
     → 사과는 빨간색이에요. 참 잘 생각했어요!
  3. (정답 O) 강아지는 '멍멍' 하고 짖을까요?
     → 맞아요! 강아지는 멍멍 하고 짖어요.

### 5단계) 완료 화면
- 좌측: "다 했어요!" + '참 잘했어요' 도장 + 획득 점수 + **이런 점이 좋아요**(말하기 자신감 / 낱말 익히기 / 즐거운 학습)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음에 할 일**(카드/점검 다시 보기 / 처음으로)

## 기술 구성
- 공용 학원 엔진(app.ts→main.js) + 과목별 data.json (수학학원과 동일 엔진)
- 의료·기업과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
