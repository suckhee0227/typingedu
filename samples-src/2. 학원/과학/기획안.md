# 기획안 — 과학 · 초등·중등: 과학 개념 확인 스마트 콘텐츠

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 과학 개념 확인 스마트 콘텐츠
- **과목/대상**: 과학 · 초등·중등
- **대상 학년/연령**: 초3 · 초4 · 초5 · 초6 · 중1
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(🔬 과학학원) + 자 진도바(✏️ 마커)
- 테마색 #2F9E44 + 포인트 #4DABF7

### 2단계) 과학 개념 카드 (6장, 플래시카드)
  1. 🧊 물질의 상태 — 고체·액체·기체로 변해요. 온도에 따라 상태가 바뀌어요.
  2. 🌱 광합성 — 식물은 빛·물·이산화탄소로 양분과 산소를 만들어요.
  3. 🚗 힘과 운동 — 힘은 물체를 밀거나 당겨요. 힘이 클수록 빠르게 움직여요.
  4. 💡 전기와 회로 — 전지·전선·전구가 이어지면 전기가 흘러 불이 켜져요.
  5. 💧 물의 순환 — 증발 → 응결 → 강수로 물은 끊임없이 돌고 돌아요.
  6. 🌑 빛과 그림자 — 빛은 곧게 나아가고, 빛이 막히면 그림자가 생겨요.

### 3단계) 학습 점검
  - 🧊 물질의 세 가지 상태를 알았어요
  - 🌱 광합성의 과정을 이해했어요
  - 🚗 힘과 운동의 관계를 알았어요
  - 💡 전기 회로의 원리를 익혔어요
  - 💧 물의 순환 과정을 기억했어요
- 우측: 💡 TIP + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 물은 100℃에서 끓어 수증기가 된다?
     → 맞아요! 물은 100℃에서 끓어 기체인 수증기가 돼요.
  2. (정답 O) 식물은 광합성으로 산소를 만든다?
     → 맞아요! 광합성으로 양분과 함께 산소를 만들어요.
  3. (정답 X) 그림자는 빛이 물체를 통과해서 생긴다?
     → 그림자는 빛이 물체에 막혀 통과하지 못할 때 생겨요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + '참 잘했어요' 도장 + 획득 점수 + **학습 효과**(개념 이해 향상 / 관찰·탐구력 향상 / 과학 자신감 상승)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드/점검 다시 보기 / 처음으로)

## 기술 구성

- 의료·기업과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
