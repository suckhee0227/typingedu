# 기획안 — 한국사·사회 · 초등·중등: 인물·사건 매칭 스마트 퀴즈

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 인물·사건 매칭 스마트 퀴즈
- **과목/대상**: 한국사·사회 · 초등·중등
- **대상 학년/연령**: 초5 · 초6 · 중1 · 중2 · 중3
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(🏯 한국사) + 자 진도바(✏️ 마커)
- 테마색 #C2410C + 포인트 #CA8A04

### 2단계) 인물·사건 카드 (6장, 플래시카드)
  1. 👑 단군왕검 — 우리 역사 최초의 국가 고조선을 세웠어요.
  2. 📜 세종대왕 — 백성을 위해 훈민정음 (한글)을 만들었어요.
  3. 🚢 이순신 — 임진왜란 때 거북선으로 나라를 지켰어요.
  4. 🏇 광개토대왕 — 고구려의 영토를 크게 넓혔어요.
  5. ⚔️ 김유신 — 신라가 삼국을 통일하는 데 큰 공을 세웠어요.
  6. 📐 정약용 — 실학자로 거중기를 만들어 수원 화성을 쌓았어요.

### 3단계) 학습 점검
  - 👑 단군왕검이 고조선을 세운 걸 알았어요
  - 📜 세종대왕이 한글을 만든 걸 알았어요
  - 🚢 이순신과 거북선을 기억했어요
  - 🏇 광개토대왕의 업적을 알았어요
  - 📐 정약용과 실학을 이해했어요
- 우측: 💡 TIP + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 세종대왕은 훈민정음(한글)을 만들었다?
     → 맞아요! 세종대왕이 백성을 위해 훈민정음을 창제했어요.
  2. (정답 X) 이순신은 우리 역사 최초의 국가 고조선을 세웠다?
     → 고조선을 세운 인물은 단군왕검이에요. 이순신은 임진왜란의 영웅이에요.
  3. (정답 O) 거북선은 임진왜란 때 사용되었다?
     → 맞아요! 이순신이 임진왜란에서 거북선으로 큰 승리를 거뒀어요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + '참 잘했어요' 도장 + 획득 점수 + **학습 효과**(역사 흐름 이해 / 인물·사건 연결력 / 한국사 자신감 상승)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드/점검 다시 보기 / 처음으로)

## 기술 구성

- 의료·기업과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
