# 기획안 — 미술학원 · 유아·초등: 미술 개념 확인 스마트 퀴즈

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 미술 개념 확인 스마트 퀴즈
- **과목/대상**: 미술학원 · 유아·초등
- **대상 학년/연령**: 유아 · 초1 · 초2 · 초3 · 초4
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(🎨 미술학원) + 자 진도바(✏️ 마커)
- 테마색 #E64980 + 포인트 #FCC419

### 2단계) 미술 개념 카드 (6장, 플래시카드)
  1. 🔴 색의 삼원색 — 빨강·파랑·노랑. 모든 색의 기본이 돼요.
  2. 🎨 색의 혼합 — 두 색을 섞으면 새로운 색이 만들어져요.
  3. 🔥 난색과 한색 — 빨강·주황은 따뜻한 색, 파랑·초록은 차가운 색.
  4. ✏️ 점·선·면 — 점이 모여 선이 되고, 선이 모여 면이 돼요.
  5. 🌗 명암 — 밝고 어두운 정도를 표현해 입체감을 줘요.
  6. 🟣 보색 — 색상환에서 마주 보는 색. 함께 쓰면 또렷해 보여요.

### 3단계) 학습 점검
  - 🔴 색의 삼원색을 알았어요
  - 🎨 색을 섞어 새 색을 만드는 걸 알았어요
  - 🔥 난색과 한색을 구분할 수 있어요
  - ✏️ 점·선·면의 관계를 이해했어요
  - 🌗 명암으로 입체감을 표현함을 알았어요
- 우측: 💡 TIP + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 빨강·파랑·노랑을 색의 삼원색이라고 한다?
     → 맞아요! 빨강·파랑·노랑이 색의 삼원색이에요.
  2. (정답 X) 파랑은 따뜻한 느낌의 난색이다?
     → 파랑은 차가운 느낌의 한색이에요. 난색은 빨강·주황이에요.
  3. (정답 O) 명암은 밝고 어두운 정도를 표현하는 것이다?
     → 맞아요! 명암으로 밝기를 표현해 입체감을 줄 수 있어요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + '참 잘했어요' 도장 + 획득 점수 + **학습 효과**(색채 감각 향상 / 관찰·표현력 향상 / 미술 자신감 상승)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드/점검 다시 보기 / 처음으로)
