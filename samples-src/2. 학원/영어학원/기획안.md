# 기획안 — 영어학원 · 초등·중등: 영어 단어 복습 스마트 카드

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 영어 단어 복습 스마트 카드
- **과목/대상**: 영어학원 · 초등·중등
- **대상 학년/연령**: 초3 · 초4 · 초5 · 초6 · 중1
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(🔤 영어학원) + 자 진도바(✏️ 마커)
- 테마색 #0CA678 + 포인트 #FAB005

### 2단계) 영어 단어 카드 (6장, 플래시카드)
  1. 🍎 apple — 뜻: 사과 예문: I eat an apple.
  2. 🦁 brave — 뜻: 용감한 예문: a brave boy.
  3. 📚 library — 뜻: 도서관 예문: Be quiet in the library.
  4. 🍽️ hungry — 뜻: 배고픈 예문: I am hungry now.
  5. 🧼 clean — 뜻: 깨끗한 / 청소하다 예문: Clean your room.
  6. ✈️ travel — 뜻: 여행하다 예문: I travel by train.

### 3단계) 학습 점검
  - 🍎 apple 의 뜻을 알았어요
  - 🦁 brave 의 뜻을 알았어요
  - 📚 library 의 뜻을 알았어요
  - 🍽️ hungry 의 뜻을 알았어요
  - ✈️ travel 의 뜻을 알았어요
- 우측: 💡 TIP + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) library 는 '도서관' 이라는 뜻이다?
     → 맞아요! library 는 '도서관'이라는 뜻이에요.
  2. (정답 X) hungry 는 '목마른' 이라는 뜻이다?
     → hungry 는 '배고픈'이에요. '목마른'은 thirsty 예요.
  3. (정답 O) brave 는 '용감한' 이라는 뜻이다?
     → 맞아요! brave 는 '용감한'이라는 뜻이에요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + '참 잘했어요' 도장 + 획득 점수 + **학습 효과**(어휘력 향상 / 발음·예문 학습 / 영어 자신감 상승)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드/점검 다시 보기 / 처음으로)

## 기술 구성
- 공용 학원 엔진(app.ts→main.js) + 과목별 data.json (수학학원과 동일 엔진)
- 의료·기업과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
