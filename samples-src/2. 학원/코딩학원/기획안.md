# 기획안 — 코딩학원 · 초등·중등: 코딩 개념 확인 스마트 퀴즈

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 코딩 개념 확인 스마트 퀴즈
- **과목/대상**: 코딩학원 · 초등·중등
- **대상 학년/연령**: 초3 · 초4 · 초5 · 초6 · 중1
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(💻 코딩학원) + 자 진도바(✏️ 마커)
- 테마색 #6741D9 + 포인트 #20C997

### 2단계) 코딩 개념 카드 (6장, 플래시카드)
  1. ➡️ 순차 — 명령을 위에서 아래로 차례대로 실행해요.
  2. 🔁 반복 — 같은 동작을 여러 번 반복해 실행해요.
  3. 🔀 조건 — '만약 ~라면'에 따라 다르게 실행해요.
  4. 📦 변수 — 값을 담아 두는 상자. 이름을 붙여 사용해요.
  5. 🧭 알고리즘 — 문제를 푸는 순서와 방법을 정한 것이에요.
  6. 🐞 디버깅 — 코드의 오류(버그)를 찾아서 고치는 일이에요.

### 3단계) 학습 점검
  - ➡️ 순차 구조를 이해했어요
  - 🔁 반복 구조를 이해했어요
  - 🔀 조건 구조를 이해했어요
  - 📦 변수가 무엇인지 알았어요
  - 🐞 디버깅의 의미를 알았어요
- 우측: 💡 TIP + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 반복(루프)은 같은 동작을 여러 번 실행하는 것이다?
     → 맞아요! 반복 구조로 같은 동작을 여러 번 실행할 수 있어요.
  2. (정답 X) 변수에는 값을 저장할 수 없다?
     → 변수는 값을 담아 두는 상자예요. 값을 저장하고 바꿀 수 있어요.
  3. (정답 O) 디버깅은 코드의 오류를 찾아 고치는 것이다?
     → 맞아요! 버그(오류)를 찾아 고치는 일을 디버깅이라고 해요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + '참 잘했어요' 도장 + 획득 점수 + **학습 효과**(논리적 사고력 향상 / 문제해결력 향상 / 코딩 자신감 상승)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드/점검 다시 보기 / 처음으로)

## 기술 구성

- 의료·기업과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
