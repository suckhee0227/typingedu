# 기획안 — 수학학원 · 초등·중등: 수학 공식·연산 체크 스마트 퀴즈

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 수학 공식·연산 체크 스마트 퀴즈
- **과목/대상**: 수학학원 · 초등·중등
- **대상 학년/연령**: 초3 · 초4 · 초5 · 초6 · 중1
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(📐 수학학원) + 자 진도바(✏️ 마커)
- 테마색 #4263EB + 포인트 #FAB005

### 2단계) 개념·공식 카드 (6장, 플래시카드)
  1. ⬛ 직사각형 넓이 — 넓이 = 가로 × 세로 단위를 꼭 함께 써요.
  2. 🔺 삼각형 넓이 — 넓이 = (밑변 × 높이) ÷ 2 높이는 밑변과 수직이에요.
  3. ⭕ 원의 둘레·넓이 — 둘레 = 지름 × 3.14 넓이 = 반지름 × 반지름 × 3.14
  4. ➗ 분수의 덧셈 — 분모가 다르면 통분한 뒤 분자끼리 더해요.
  5. ⚖️ 비와 비율 — 비율 = 비교하는 양 ÷ 기준량 백분율은 비율 × 100.
  6. 🔢 연산 순서 — 괄호 → 곱셈·나눗셈 → 덧셈·뺄셈 순으로 계산해요.

### 3단계) 학습 점검
  - 🔺 삼각형 넓이 공식을 이해했어요
  - ⭕ 원의 둘레·넓이를 알았어요
  - ➗ 분수는 통분 후 더하는 걸 알았어요
  - ⚖️ 비와 비율 구하는 법을 익혔어요
  - 🔢 연산 순서를 기억했어요
- 우측: 💡 TIP + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 삼각형의 넓이는 (밑변 × 높이) ÷ 2 이다?
     → 맞아요! 삼각형 넓이는 (밑변 × 높이) ÷ 2 로 구해요.
  2. (정답 X) 분수의 덧셈은 분모가 달라도 분자끼리 그냥 더한다?
     → 분모가 다르면 먼저 통분한 뒤 분자끼리 더해야 해요.
  3. (정답 O) 곱셈·나눗셈은 덧셈·뺄셈보다 먼저 계산한다?
     → 맞아요! 곱셈·나눗셈을 덧셈·뺄셈보다 먼저 계산해요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + '참 잘했어요' 도장 + 획득 점수 + **학습 효과**(연산 정확도 향상 / 개념 정리 / 수학 자신감 상승)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드/점검 다시 보기 / 처음으로)
