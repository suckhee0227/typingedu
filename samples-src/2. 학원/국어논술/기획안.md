# 기획안 — 국어·논술 · 초등·중등: 어휘·문장 이해 스마트 퀴즈

> 학원 교육용 라인업. 의료·기업과 **다른 새 "스터디 노트" UI**(모눈 배경, 공책 인덱스 탭,
> 자(ruler) 진도바, 노트 페이지 패널+빨간 마진선, 플래시카드, '참 잘했어요' 도장)로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 어휘·문장 이해 스마트 퀴즈
- **과목/대상**: 국어·논술 · 초등·중등
- **대상 학년/연령**: 초3 · 초4 · 초5 · 초6 · 중1
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 대상 학년 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 목표**(6항목) + **이용 안내** 패널
- 상단: 공책 인덱스 탭(📖 국어·논술) + 자 진도바(✏️ 마커)
- 테마색 #1864AB + 포인트 #E8590C

### 2단계) 어휘·표현 카드 (6장, 플래시카드)
  1. 🤔 신중하다 — 뜻: 매우 조심스럽게 생각하고 행동하다.
  2. 💗 공감 — 뜻: 남의 감정이나 생각을 함께 느끼는 것.
  3. 🎭 비유 — 어떤 것을 다른 것에 빗대어 표현하는 방법.
  4. 📝 요약 — 글의 중요한 내용만 짧게 간추리는 것.
  5. ⚖️ 주장과 근거 — 주장은 내 의견, 근거는 그렇게 생각한 까닭.
  6. 🗣️ 속담 — 조상의 지혜와 교훈을 담은 짧은 말.

### 3단계) 학습 점검
  - 🤔 '신중하다'의 뜻을 알았어요
  - 💗 '공감'의 뜻을 알았어요
  - 🎭 비유가 무엇인지 이해했어요
  - 📝 글을 요약하는 법을 익혔어요
  - ⚖️ 주장에는 근거가 필요함을 알았어요
- 우측: 💡 TIP + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) '신중하다'는 '조심스럽게 생각하고 행동하다'라는 뜻이다?
     → 맞아요! '신중하다'는 조심스럽게 생각하고 행동하는 것을 뜻해요.
  2. (정답 X) 비유는 사실을 있는 그대로만 설명하는 방법이다?
     → 비유는 어떤 것을 다른 것에 빗대어 표현하는 방법이에요.
  3. (정답 O) 주장을 할 때에는 근거가 필요하다?
     → 맞아요! 주장에는 그렇게 생각한 근거가 있어야 설득력이 생겨요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + '참 잘했어요' 도장 + 획득 점수 + **학습 효과**(어휘력·독해력 향상 / 표현력 향상 / 국어 자신감 상승)
- 우측: **학원 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드/점검 다시 보기 / 처음으로)

## 기술 구성

- 의료·기업과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
