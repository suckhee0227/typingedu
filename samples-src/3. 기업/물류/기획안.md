# 기획안 — 물류 · 배송업체: 물류센터 안전수칙 스마트 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 물류센터 안전수칙 스마트 퀴즈
- **분야/업종**: 물류 · 배송업체
- **적용 가능 업종**: 물류센터 · 배송업체 · 냉동·냉장 · 이커머스 · 제조·유통 · 3PL 업체
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🦺 안전 교육) + 5단계 스텝 인디케이터
- **레이아웃**: `split` (좌우 분할형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #EA7317 + 포인트 #2563EB

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. ⚠️ 주요 위험 요소 — 낙하·끼임·충돌 등 현장의 위험 요소를 파악해요.
  2. 🦺 보호구 착용 — 안전화·안전모·장갑을 작업 전 반드시 착용해요.
  3. 🚜 지게차 안전수칙 — 지게차 통행로와 신호를 지키며 작업해요.
  4. 📦 운반·적재 안전 — 올바른 자세로 들고 정해진 방식으로 적재해요.
  5. 🧯 비상상황 대처 — 화재·사고 시 대피로와 초기 대응 방법을 익혀요.
  6. 📋 작업 전 점검 — 장비·구역 상태를 작업 전 미리 점검해요.

### 3단계) 물류센터 안전 체크리스트
  - 🦺 작업 전 보호구를 착용했어요
  - 🚜 지게차 통행로를 확인했어요
  - 📦 정해진 방식으로 적재했어요
  - 🧯 비상구·소화기 위치를 확인했어요
  - 💪 무리한 중량물을 혼자 들지 않았어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 지게차와 보행자가 같은 통로를 사용해도 안전하다?
     → 충돌 위험이 커요. 지정 통행로를 분리해 이용하세요.
  2. (정답 O) 무거운 물건은 허리보다 다리 힘으로 들어야 한다?
     → 허리 부상을 막으려면 무릎을 굽혀 다리 힘으로 드세요.
  3. (정답 O) 비상구 앞에는 물건을 쌓아두면 안 된다?
     → 대피로 확보는 필수예요. 비상구 앞은 항상 비워두세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(산업재해 예방 / 안전한 작업 환경 / 작업 효율 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
