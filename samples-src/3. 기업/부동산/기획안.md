# 기획안 — 부동산중개업: 계약 전 체크리스트 스마트 안내

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 계약 전 체크리스트 스마트 안내
- **분야/업종**: 부동산중개업
- **적용 가능 업종**: 부동산중개 · 아파트·빌라 · 상가·오피스 · 전·월세 · 토지·공장 · 분양·관리
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🏠 중개 교육) + 5단계 스텝 인디케이터
- **레이아웃**: `sidebar` (좌측 사이드바형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #1E40AF + 포인트 #C8A04B

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 📜 등기부등본 확인 — 소유권·근저당 등 권리 사항을 계약 전 반드시 확인해요.
  2. ⚖️ 권리관계 점검 — 가압류·제한물권 등 권리관계를 꼼꼼히 점검해요.
  3. ✍️ 계약서 특약사항 — 구두 약속은 반드시 특약으로 계약서에 명확히 기재해요.
  4. 💰 중개보수·비용 — 중개보수와 부대 비용을 거래 전에 미리 안내해요.
  5. 🔒 고객 정보 보호 — 개인정보와 계약 정보를 안전하게 관리해요.
  6. ⚠️ 주의사항 안내 — 계약 시 유의할 점을 안내해 분쟁을 예방해요.

### 3단계) 계약 전 체크리스트
  - 📜 등기부등본을 확인했어요
  - ⚖️ 권리관계·근저당을 확인했어요
  - ✍️ 계약서 특약사항을 점검했어요
  - 💰 중개보수를 미리 안내했어요
  - 🪪 본인·대리 여부를 확인했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 계약 전에 등기부등본을 반드시 확인해야 한다?
     → 소유권·권리관계 확인은 기본이에요. 계약 전 꼭 확인하세요.
  2. (정답 X) 계약서 특약사항은 구두로만 약속해도 된다?
     → 구두 약속은 분쟁의 원인이 돼요. 반드시 특약으로 명시하세요.
  3. (정답 O) 중개보수는 거래 전에 미리 안내해야 한다?
     → 비용을 사전에 안내하면 신뢰와 투명성이 높아져요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(계약 안전성 향상 / 중개 신뢰도 상승 / 분쟁 예방)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
