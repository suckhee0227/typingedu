# 기획안 — 식품제조 · 소규모 공장: 식품위생·작업안전 스마트 안내

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 식품위생·작업안전 스마트 안내
- **분야/업종**: 식품제조 · 소규모 공장
- **적용 가능 업종**: 식품제조 · 베이커리 · 반찬·도시락 · 유가공 · 가공식품 · 소규모 공장
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🥗 위생 교육) + 5단계 스텝 인디케이터
- **레이아웃**: `split` (좌우 분할형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #2E9E5B + 포인트 #F59E0B

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 🧼 개인위생 관리 — 손 씻기·복장·건강 상태를 작업 전 꼼꼼히 관리해요.
  2. 🌡️ 식품보관 기준 — 보관 온도와 유통기한을 기준에 맞게 관리해요.
  3. 🔪 교차오염 예방 — 식자재와 도구를 용도별로 구분해 사용해요.
  4. 🧹 작업장 정돈 — 청결과 정리정돈으로 위생 환경을 유지해요.
  5. 🧤 작업안전 수칙 — 장비 사용법과 안전 수칙을 반드시 지켜요.
  6. 🚨 사고 예방·대처 — 위험 요소를 미리 살피고 사고 시 대응 방법을 익혀요.

### 3단계) 위생·안전 실전 체크리스트
  - 🧼 작업 전 손을 깨끗이 씻었어요
  - 🧤 위생 복장·장갑을 착용했어요
  - 🌡️ 식자재 보관 온도를 확인했어요
  - 🔪 도구를 용도별로 구분해 사용했어요
  - ✨ 작업장을 청결하게 정돈했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 손 씻기는 식품 위생 관리의 가장 기본이다?
     → 손 위생은 식품 안전의 출발점이에요. 작업 전·후 꼭 씻어주세요.
  2. (정답 X) 익힌 식품과 날 식품은 같은 도구로 다뤄도 된다?
     → 교차오염 위험이 있어요. 도구를 구분해 사용하세요.
  3. (정답 O) 작업장 바닥의 물기는 바로 닦아야 한다?
     → 미끄럼 사고를 막으려면 물기는 즉시 닦아주세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(식품 안전성 확보 / 위생 사고 예방 / 작업 효율 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
