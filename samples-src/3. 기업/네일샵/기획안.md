# 기획안 — 네일샵: 네일 시술 후 관리 안내 스마트 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 네일 시술 후 관리 안내 스마트 퀴즈
- **분야/업종**: 네일샵
- **적용 가능 업종**: 네일샵 · 젤네일 전문 · 네일&페디 · 프랜차이즈 · 네일 용품점 · 뷰티샵
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **이런 내용을 배워요**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(💅 네일 케어) + 5단계 스텝 인디케이터
- **레이아웃**: `hero` (상단 히어로 배너형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #C062A6 + 포인트 #F4B8D0

### 2단계) 핵심 관리 카드 (6장 스와이프)
  1. ⏳ 시술 후 2~3시간 주의 — 완전히 굳을 때까지 충격을 주지 않아요.
  2. 🖐️ 손톱 보호 — 손톱을 무리하게 사용하지 않아요.
  3. 💧 보습 관리 — 큐티클과 손을 자주 보습해요.
  4. 🧤 생활 속 주의 — 설거지·청소할 때는 장갑을 착용해요.
  5. 🚫 손상 예방 — 네일을 도구처럼 사용하지 않아요.
  6. 📅 재방문 관리 — 3~4주 간격으로 정기 관리를 받아요.

### 3단계) 네일 관리 체크리스트
  - ⏳ 시술 후 2~3시간 충격을 주지 않았어요
  - 🖐️ 손톱을 도구처럼 사용하지 않았어요
  - 💧 큐티클·손 보습을 했어요
  - 🧤 물·세제 작업 시 장갑을 착용했어요
  - 📅 3~4주 관리 주기를 확인했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 시술 직후 바로 설거지를 해도 괜찮다?
     → 굳기 전 물·세제는 손상의 원인! 장갑을 끼거나 시간을 두세요.
  2. (정답 O) 손·큐티클 보습은 네일 유지에 도움이 된다?
     → 보습은 들뜸을 막아 네일을 오래 유지해줘요.
  3. (정답 O) 네일을 도구처럼 사용하면 손상되기 쉽다?
     → 긁고 뜯는 사용은 손상의 원인! 손톱은 손톱으로만 써주세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(네일 유지 기간 향상 / 손톱 건강 보호 / 재방문율 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
