# 기획안 — 콜센터/상담업체: 상담원 응대 매뉴얼 스마트 퀴즈

> 기업 교육용 신규 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과,
> 회사 연락처)으로 재구성했다.

## 콘텐츠 개요
- **콘텐츠명**: 상담원 응대 매뉴얼 스마트 퀴즈
- **대상**: 콜센터·고객상담 신입/재직 상담원 (1인용)
- **적용 가능 업종**: 콜센터 · 고객상담팀 · 온라인고객센터 · 금융/보험 · 여행/항공 · 통신/인터넷
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **콘텐츠 의도**: 핵심 학습 카드 → 실전 체크리스트 → 확인 퀴즈로 이어지는 단계형 구성으로,
  상담 상황별 응대 방법을 학습하고 고객 만족도를 높이는 전문 상담원으로 성장하도록 돕는다.
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 콘텐츠 제목·소개 문구 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내**(4항목) 패널
- 상단: 브랜드 칩(🎧 상담 교육) + 5단계 스텝 인디케이터
- 분위기: 신뢰감 있는 코퍼릿 톤(테마색 #6C4FD8 + 포인트 #19C2A8)

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 🙌 기본 응대 매너 — 밝은 인사·끝까지 경청·정중한 말투
  2. 🗂️ 상담 단계별 응대 — 인사 → 용건 확인 → 안내 → 마무리
  3. 🤝 불만 고객 응대 — 먼저 공감·사과 후 해결 방안 안내
  4. 🔒 개인정보 보호 — 본인 확인 절차·정보 보안 수칙 준수
  5. 📋 정확한 정보 제공 — 확인되지 않은 내용은 단정하지 않기
  6. ✅ 마무리 및 후속조치 — 처리 내용 확인 + 후속 조치 안내
- 카드별 아이콘·제목·설명, 도트 인디케이터, 이전/다음 네비게이션

### 3단계) 응대 실전 체크리스트
- 좌측: 직접 체크하는 5개 항목(모두 체크해야 다음 단계 이동) + 진행 바
  - 👂 고객의 말을 끝까지 경청했어요
  - 💗 공감과 적절한 호응을 표현했어요
  - 📋 정확한 정보를 안내했어요
  - 🔒 개인정보 보호 절차를 지켰어요
  - 📞 통화 마무리 인사를 했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 고객의 말을 끊고 빠른 해결책을 먼저 제시해야 한다?
     → 먼저 끝까지 경청·공감한 뒤 해결 방안을 안내합니다.
  2. (정답 O) 개인정보는 반드시 본인 확인 후에 처리해야 한다?
     → 본인 확인은 정보 보호의 기본입니다.
  3. (정답 O) 통화를 마무리할 때 처리 내용을 다시 확인해야 한다?
     → 마무리 확인은 오해를 줄이고 고객 만족을 높입니다.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(상담 품질 향상 / 고객 만족도 상승 / 신입 교육 효율화)
- 우측: **문의 및 안내**(전화·카카오채널·인스타) + **다음 행동 선택**(핵심 카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 디자인 가이드 (기업용 새 엔진)

- 배경: 은은한 도트 그리드 + 떠다니는 둥근 사각형(코퍼릿 모션) — 의료판의 물방울 모티브 대체
- 점수제(+10점)·기대효과·회사 연락처로 기업 교육 맥락 강화
- 16:9 가로 화면 최적화, 손가락으로 누르기 쉬운 큰 버튼

## 기술 구성
- 공용 데이터 기반 엔진(app.ts) + 업종별 data.json
- 의료판과 **동일한 패키징 구조**(index.html / main.js / style.css / data.json / appHelper.ts)
- 단계 전환은 화면 내에서 자연스럽게 처리, 별도 외부 라이브러리 의존 없음
- data.json 스키마 확장: learnList / usageList / industries / tip / effects / contact(kakao·instagram) / quizPointLabel
