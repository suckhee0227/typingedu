# 기획안 — 온라인쇼핑몰: 고객 혜택 찾기 스마트 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 고객 혜택 찾기 스마트 퀴즈
- **분야/업종**: 온라인쇼핑몰
- **적용 가능 업종**: 패션·의류 · 뷰티·화장품 · 식품·건강 · 생활·잡화 · 종합몰 · 오픈마켓
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🛍️ 쇼핑 혜택) + 5단계 스텝 인디케이터
- **레이아웃**: `hero` (상단 히어로 배너형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #EC4899 + 포인트 #FB923C

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 🎟️ 쿠폰 & 할인 — 쿠폰을 발급받아 적용하면 더 저렴하게 구매할 수 있어요.
  2. 🚚 무료 배송 조건 — 기준 금액 이상 주문하면 무료 배송 혜택을 받아요.
  3. 🅿️ 포인트 적립·사용 — 구매할 때마다 적립금을 모아 다음 결제에 사용해요.
  4. 👑 회원 등급 혜택 — 등급이 오를수록 추가 할인과 특별 혜택이 늘어나요.
  5. 🔄 교환·반품 정책 — 교환·반품 기준과 기간을 미리 확인하면 안심하고 구매해요.
  6. 🔒 안전한 결제 — 공식 결제 수단을 이용하고 비밀번호를 안전하게 관리해요.

### 3단계) 혜택 활용 체크리스트
  - 🎟️ 받을 수 있는 쿠폰을 적용했어요
  - 🚚 무료 배송 조건을 확인했어요
  - 🅿️ 적립금을 확인하고 사용했어요
  - 👑 내 회원 등급 혜택을 확인했어요
  - 🔒 안전한 결제 수단을 이용했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 쿠폰은 상품에 따라 중복 사용이 제한될 수 있다?
     → 쿠폰별 사용 조건이 달라요. 적용 전에 조건을 확인하세요.
  2. (정답 O) 무료 배송 기준 금액 미만이면 배송비가 부과될 수 있다?
     → 기준 금액 이상일 때 무료 배송! 조금 더 담아 혜택을 받는 것도 방법이에요.
  3. (정답 X) 교환·반품은 기간 제한 없이 언제든 가능하다?
     → 보통 수령 후 일정 기간 내에만 가능해요. 정책과 기간을 꼭 확인하세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(혜택 활용도 향상 / 고객 만족도 상승 / 재구매율 증가)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
