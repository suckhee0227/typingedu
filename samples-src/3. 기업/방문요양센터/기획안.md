# 기획안 — 방문요양센터: 방문요양 안전수칙 스마트 안내

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 방문요양 안전수칙 스마트 안내
- **분야/업종**: 방문요양센터
- **적용 가능 업종**: 방문요양 · 노인 요양 · 재가 케어 · 복지서비스 · 요양기관 · 요양보호사 교육
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🤲 요양 교육) + 5단계 스텝 인디케이터
- **레이아웃**: `bottombar` (하단바형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #2EA98C + 포인트 #F0913E

### 2단계) 핵심 안전수칙 카드 (6장 스와이프)
  1. 🧼 손 위생·감염 예방 — 방문 전후 손 위생과 감염 예방 수칙을 지켜요.
  2. 🛟 어르신 안전 관리 — 낙상 등 안전 사고를 미리 살피고 예방해요.
  3. 🏠 낙상 위험 점검 — 문턱·미끄러운 바닥 등 위험 환경을 점검해요.
  4. 🚶 이동·보행 보조 — 올바른 자세로 안전하게 이동과 보행을 보조해요.
  5. 🚑 응급상황 대처 — 이상 징후 시 침착하게 대응하고 신속히 알려요.
  6. 💊 복약·기록 관리 — 복약 시간·용량을 지키고 서비스 내용을 기록해요.

### 3단계) 업무 전 안전 체크리스트
  - 🧼 손 위생·보호장구를 착용했어요
  - ❤️ 어르신 건강 상태를 확인했어요
  - 🏠 낙상 위험 요소를 점검했어요
  - 💊 복약 시간·용량을 확인했어요
  - 📝 서비스 내용을 기록했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 어르신이 어지러워하면 바로 일으켜 걷게 해야 한다?
     → 낙상 위험이 있어요. 안정시킨 뒤 천천히 도와야 해요.
  2. (정답 O) 복약은 정해진 시간과 용량을 지켜야 한다?
     → 복약 관리는 건강의 기본이에요. 시간·용량을 꼭 지켜주세요.
  3. (정답 O) 응급상황 시 기관에 보고하고 가족에게 연락해야 한다?
     → 신속한 보고와 연락이 어르신의 안전을 지켜요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(어르신 안전 보호 / 안전 사고 예방 / 케어 품질 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
