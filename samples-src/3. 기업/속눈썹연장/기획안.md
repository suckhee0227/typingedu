# 기획안 — 속눈썹샵: 속눈썹 연장 후 관리 안내 스마트 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 속눈썹 연장 후 관리 안내 스마트 퀴즈
- **분야/업종**: 속눈썹샵
- **적용 가능 업종**: 속눈썹샵 · 뷰티샵 · 연장·미용 · 메이크업샵 · 네일&뷰티 · 프랜차이즈
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(👁️ 속눈썹 케어) + 5단계 스텝 인디케이터
- **레이아웃**: `minimal` (미니멀 중앙형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #8E5CC0 + 포인트 #E0A6D8

### 2단계) 핵심 관리 카드 (6장 스와이프)
  1. ⏳ 연장 후 24시간 — 24시간 동안 물과 습기를 피해주세요.
  2. 💦 물 접촉 주의 — 사우나·수영·찜질은 며칠간 자제해요.
  3. 🧼 세안 방법 — 비비지 않고 부드럽게 세안해요.
  4. 🚫 관리 제품 — 오일 성분 제품은 접촉을 피해요.
  5. 🖐️ 손상 예방 — 비비거나 잡아당기지 않아요.
  6. 📅 재방문 주기 — 2~3주 간격 리터치로 풍성하게 유지해요.

### 3단계) 속눈썹 관리 체크리스트
  - ⏳ 연장 후 24시간 물을 피했어요
  - 💦 사우나·수영을 자제했어요
  - 🧼 비비지 않고 부드럽게 세안했어요
  - 🚫 오일 성분 제품을 피했어요
  - 📅 2~3주 리터치 주기를 확인했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 연장 후 24시간 이내에 물에 닿아도 괜찮다?
     → 접착이 굳기 전 물은 금물! 24시간은 물을 피해주세요.
  2. (정답 O) 오일 성분 제품은 속눈썹 유지에 영향을 줄 수 있다?
     → 오일은 접착력을 약하게 해요. 오일프리 제품을 사용하세요.
  3. (정답 O) 속눈썹을 자주 비비면 손상되기 쉽다?
     → 비비고 잡아당기면 빠지기 쉬워요. 부드럽게 다뤄주세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(유지 기간 향상 / 속눈썹 손상 예방 / 재방문율 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
