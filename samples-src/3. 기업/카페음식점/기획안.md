# 기획안 — 카페 · 음식점: 신입 알바 매장교육 스마트 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 신입 알바 매장교육 스마트 퀴즈
- **분야/업종**: 카페 · 음식점
- **적용 가능 업종**: 카페 · 베이커리 · 분식·식당 · 샐러드·도시락 · 프랜차이즈 · 디저트
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 교육 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(☕ 매장 교육) + 5단계 스텝 인디케이터
- **레이아웃**: `split` (좌우 분할형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #8B5E3C + 포인트 #C8923B

### 2단계) 핵심 업무 카드 (5장 스와이프)
  1. 🔆 오픈 준비 — 매장 오픈 전 청소·세팅·재료 준비 절차를 순서대로 익혀요.
  2. 🧾 주문 응대 — 밝은 인사와 정확한 주문 확인으로 친절하게 응대해요.
  3. 🥤 메뉴 제조 — 정해진 레시피와 용량대로 일정한 맛을 유지해요.
  4. 🙂 고객 응대 — 요청과 컴플레인에 침착하게 응대해 만족도를 높여요.
  5. 🧹 마감 정리 — 청결·재고·마감 점검으로 다음 영업을 준비해요.

### 3단계) 위생 & 안전 체크리스트
  - 🧼 손 위생과 복장 청결을 지켰어요
  - 🌡️ 식자재 보관 온도를 확인했어요
  - 🍳 조리 도구를 위생적으로 사용했어요
  - ✨ 매장 청결 상태를 유지했어요
  - ⚠️ 안전 수칙을 준수했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 유통기한이 지난 식자재도 겉이 멀쩡하면 사용해도 된다?
     → 기한이 지난 식자재는 사용 금지! 폐기하고 새 재료를 사용하세요.
  2. (정답 O) 손 씻기는 매장 위생 관리의 가장 기본이다?
     → 손 위생은 식품 안전의 기본이에요. 근무 전·후 꼭 씻어주세요.
  3. (정답 O) 마감 시 재고와 청결 상태를 확인해야 한다?
     → 마감 점검은 다음 영업의 시작이에요. 재고·청결을 꼭 확인하세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(신입 적응 향상 / 매장 운영 효율 / 서비스 품질 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
