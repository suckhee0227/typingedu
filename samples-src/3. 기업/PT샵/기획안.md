# 기획안 — PT샵 · 필라테스: 첫 수업 준비 스마트 안내

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 첫 수업 준비 스마트 안내
- **분야/업종**: PT샵 · 필라테스
- **적용 가능 업종**: PT샵 · 필라테스 · 요가원 · 헬스장 · 피트니스 · 복싱·크로스핏
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **이런 내용을 배워요**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(💪 운동 가이드) + 5단계 스텝 인디케이터
- **레이아웃**: `minimal` (미니멀 중앙형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #EF4444 + 포인트 #F59E0B

### 2단계) 핵심 준비 카드 (6장 스와이프)
  1. 👕 운동 복장 준비 — 활동이 편한 복장과 운동화를 준비해요.
  2. 🎒 준비물 챙기기 — 수건·물병 등 필요한 준비물을 미리 챙겨요.
  3. 💧 수분 섭취 — 운동 전후 수분을 충분히 보충해요.
  4. 🤸 스트레칭 — 운동 전후 스트레칭으로 부상을 예방해요.
  5. ⏱️ 운동 시간·강도 — 무리하지 않는 강도로 천천히 늘려가요.
  6. ❤️ 부상 예방 — 통증이 느껴지면 무리하지 않아요.

### 3단계) 첫 수업 준비 체크리스트
  - 👟 운동 복장·신발을 준비했어요
  - 🎒 수건·물병을 챙겼어요
  - 🤸 준비운동·스트레칭을 했어요
  - ⏰ 수업 시작 10분 전 도착했어요
  - ❤️ 몸 상태(통증·컨디션)를 확인했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 수업 직전 2시간 이내에 과식하는 것이 좋다?
     → 과식은 운동에 방해가 돼요. 수업 전 식사는 가볍게 하세요.
  2. (정답 O) 운동 전후 스트레칭은 부상 예방에 도움이 된다?
     → 스트레칭은 근육을 풀어 부상을 예방해줘요.
  3. (정답 O) 통증이 느껴지면 무리하지 말고 트레이너에게 알려야 한다?
     → 통증은 몸의 신호예요. 무리하지 말고 바로 알리세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(부상 예방 / 운동 효과 향상 / 수업 만족도 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성

- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
