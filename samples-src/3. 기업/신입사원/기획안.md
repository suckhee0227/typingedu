# 기획안 — 신입사원 온보딩: 신입사원 온보딩 스마트 안내 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 신입사원 온보딩 스마트 안내 퀴즈
- **분야/업종**: 신입사원 온보딩
- **적용 가능 업종**: 기업·회사 · 교육기관 · 의료기관 · 프랜차이즈 · 공공기관 · 제조·물류
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🏢 온보딩) + 5단계 스텝 인디케이터
- **레이아웃**: `sidebar` (좌측 사이드바형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #4F46E5 + 포인트 #22C55E

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 🏷️ 회사 소개 — 비전·미션·핵심 가치를 이해하고 소속감을 키워요.
  2. 🗂️ 조직과 업무 — 조직도와 내 담당 업무, 협업 구조를 파악해요.
  3. 🗓️ 근무 규정 — 근태·휴가·복무 규정을 정확히 익혀요.
  4. 💝 복지 제도 — 복지와 지원 제도를 확인해 적극 활용해요.
  5. 🔒 보안 & 윤리 — 정보 보안과 윤리 기준을 반드시 지켜요.
  6. 💻 시스템·프로세스 — 사내 시스템과 업무 프로세스 사용법을 익혀요.

### 3단계) 온보딩 실전 체크리스트
  - 🏷️ 회사 비전·가치를 이해했어요
  - 🗂️ 조직도와 담당 업무를 파악했어요
  - 🗓️ 근무 규정·근태 방식을 확인했어요
  - 🔒 보안·정보 보호 수칙을 숙지했어요
  - 💻 사내 시스템 사용법을 익혔어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 회사의 비밀 정보는 외부에 자유롭게 공유해도 된다?
     → 기밀 정보는 외부 공유 금지! 보안 규정을 반드시 지켜주세요.
  2. (정답 O) 연차 휴가는 입사 후 규정에 따라 사용할 수 있다?
     → 근로기준과 사내 규정에 따라 연차를 사용할 수 있어요.
  3. (정답 O) 보안·정보 보호 수칙은 의무적으로 지켜야 한다?
     → 보안은 선택이 아닌 의무예요. 작은 수칙부터 지켜주세요.

### 5단계) 완료 화면
- 좌측: "온보딩 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(신입 적응 향상 / 조직 이해도 향상 / 온보딩 교육 효율화)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
