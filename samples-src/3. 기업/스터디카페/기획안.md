# 기획안 — 스터디카페 · 무인매장: 이용수칙 스마트 안내

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 이용수칙 스마트 안내
- **분야/업종**: 스터디카페 · 무인매장
- **적용 가능 업종**: 스터디카페 · 무인카페 · 무인운영점 · 공유오피스 · 24시 편의시설 · 코워킹스페이스
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **이런 내용을 확인해요**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(📚 이용 안내) + 5단계 스텝 인디케이터
- **레이아웃**: `topbar` (상단바형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #4C6EF5 + 포인트 #38B2AC

### 2단계) 핵심 이용수칙 카드 (6장 스와이프)
  1. 🚪 출입·이용 방법 — 키오스크 결제와 출입 방법을 익혀요.
  2. 🪑 좌석 이용 — 지정 좌석을 이용하고 사용 시간을 지켜요.
  3. 🤫 정숙 매너 — 통화·소음을 자제하고 조용히 이용해요.
  4. 🍱 음식물 반입 — 허용·금지 음식을 확인해 규정을 지켜요.
  5. ⏰ 이용 시간 관리 — 이용 시간과 연장 방법을 확인해요.
  6. 📹 CCTV·보안 — 무인 매장도 CCTV로 안전하게 관리돼요.

### 3단계) 이용 다짐 체크리스트
  - 🚪 출입·결제 방법을 확인했어요
  - 🪑 지정 좌석을 이용했어요
  - 🤫 정숙 매너를 지켰어요
  - 🍱 음식물 반입 규정을 지켰어요
  - 🧹 퇴실 시 자리를 정리했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 스터디카페 안에서 큰 소리로 통화해도 된다?
     → 정숙은 기본 매너예요. 통화는 지정 공간에서 해주세요.
  2. (정답 O) 사용한 자리는 퇴실 시 정리해야 한다?
     → 다음 이용자를 위해 자리를 깨끗이 정리해주세요.
  3. (정답 O) 무인매장도 CCTV로 안전하게 관리된다?
     → CCTV로 안전을 관리하니 안심하고 이용하세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(쾌적한 이용 환경 / 분쟁·민원 감소 / 매장 관리 효율화)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
