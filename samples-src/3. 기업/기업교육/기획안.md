# 기획안 — 기업교육 · 고객서비스: 고객 서비스 역량 강화 스마트 상황 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 고객 서비스 역량 강화 스마트 상황 퀴즈
- **분야/업종**: 기업교육 · 고객서비스
- **적용 가능 업종**: 유통·매장 · 콜센터 · 외식·서비스 · 교육·학원 · 금융·보험 · 의료기관
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🎓 서비스 교육) + 5단계 스텝 인디케이터
- **레이아웃**: `topbar` (상단바형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #2563EB + 포인트 #14B8A6

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 🤝 서비스 기본 원칙 — 고객 응대의 기본 원칙으로 첫인상의 신뢰를 만들어요.
  2. 👂 경청과 공감 — 고객의 말을 끝까지 듣고 진심으로 공감해요.
  3. ℹ️ 요청·문의 응대 — 요청과 문의를 정확하고 신속하게 처리해요.
  4. 🙂 불만 고객 응대 — 감정에 휘둘리지 않고 침착하게 해결책을 안내해요.
  5. 🧭 어려운 상황 대처 — 돌발·난처한 상황을 차분하고 유연하게 대응해요.
  6. 💙 서비스 마인드 — 고객 중심으로 생각하는 태도를 몸에 익혀요.

### 3단계) 서비스 실전 체크리스트
  - 🙌 밝게 인사하고 응대했어요
  - 👂 고객의 말을 끝까지 경청했어요
  - ℹ️ 정확하게 정보를 전달했어요
  - 💗 불만에 침착하게 대응했어요
  - 🧭 고객 입장에서 생각했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 고객의 불만에는 변명부터 하는 것이 좋다?
     → 먼저 경청하고 공감한 뒤 해결책을 안내하는 것이 좋아요.
  2. (정답 O) 고객의 요청은 정확히 확인한 후 처리해야 한다?
     → 정확한 확인은 실수를 줄이고 신뢰를 높여줘요.
  3. (정답 O) 서비스 마인드는 교육으로 향상시켜 나갈 수 있다?
     → 꾸준한 학습과 실천으로 서비스 역량은 성장해요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(응대 역량 향상 / 서비스 품질 표준화 / 브랜드 이미지 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
