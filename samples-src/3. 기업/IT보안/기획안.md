# 기획안 — IT 보안: 정보보안 습관 만들기 스마트 퀴즈

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 정보보안 습관 만들기 스마트 퀴즈
- **분야/업종**: IT 보안
- **적용 가능 업종**: IT기업 · 공공기관 · 금융 · 이커머스 · 의료기관 · 교육기관
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🔐 정보보안) + 5단계 스텝 인디케이터
- **레이아웃**: `sidebar` (좌측 사이드바형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #1E3A8A + 포인트 #06B6D4

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 🔑 강력한 비밀번호 — 길고 복잡하게 설정하고 사이트마다 다르게 관리해요.
  2. 🎣 피싱·스미싱 주의 — 의심스러운 링크·문자·메일은 클릭하지 않아요.
  3. 🪪 개인정보 보호 — 꼭 필요한 정보만 입력하고 화면을 잠그고 관리해요.
  4. 🛡️ 악성코드 차단 — 백신과 보안 업데이트를 항상 최신으로 유지해요.
  5. 📶 안전한 Wi-Fi — 공용 와이파이에서 중요한 업무·결제는 피해요.
  6. 🚨 보안사고 대응 — 이상 징후는 즉시 신고하고 초기 대응 절차를 따라요.

### 3단계) 정보보안 실전 체크리스트
  - 🔑 비밀번호를 주기적으로 변경했어요
  - 🔐 2단계(OTP) 인증을 사용했어요
  - 🎣 의심스러운 링크를 클릭하지 않았어요
  - 🛡️ 백신·보안 업데이트를 실행했어요
  - 🖥️ 자리를 비울 때 화면을 잠갔어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 여러 사이트에 같은 비밀번호를 사용해도 안전하다?
     → 하나가 유출되면 전부 위험해요. 사이트마다 다르게 설정하세요.
  2. (정답 O) 출처가 불분명한 메일 첨부파일은 열지 않는 것이 좋다?
     → 악성코드 감염의 주요 경로예요. 의심되면 열지 마세요.
  3. (정답 O) 보안사고 발생 시 즉시 담당자에게 신고해야 한다?
     → 빠른 신고가 피해를 줄여요. 숨기지 말고 바로 알리세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(보안 사고 예방 / 정보 보호 강화 / 보안 의식 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
