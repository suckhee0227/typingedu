# 기획안 — 뷰티샵 · 에스테틱: 시술 후 홈케어 스마트 안내

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 시술 후 홈케어 스마트 안내
- **분야/업종**: 뷰티샵 · 에스테틱
- **적용 가능 업종**: 피부관리 · 에스테틱 · 왁싱 · 두피관리 · 뷰티샵 · 스파
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(💆 홈케어 안내) + 5단계 스텝 인디케이터
- **레이아웃**: `hero` (상단 히어로 배너형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #E0719C + 포인트 #F0A6C0

### 2단계) 핵심 케어 카드 (6장 스와이프)
  1. 🚫 시술 직후 주의 — 시술 부위를 만지거나 자극하지 않아요.
  2. 💧 세안 & 보습 — 순한 제품으로 세안하고 충분히 보습해요.
  3. ☀️ 자외선 차단 — 외출 시 자외선 차단제를 꼼꼼히 발라요.
  4. 🧖 피해야 할 행동 — 사우나·과한 운동·음주는 일정 기간 피해요.
  5. 🧴 홈케어 제품 — 추천받은 제품을 올바른 순서로 사용해요.
  6. 📅 재방문 관리 — 다음 관리 주기를 지켜 효과를 유지해요.

### 3단계) 홈케어 실전 체크리스트
  - 🚫 시술 부위를 자극하지 않았어요
  - 💧 순한 제품으로 세안·보습했어요
  - ☀️ 자외선 차단제를 발랐어요
  - 🧖 사우나·음주를 피했어요
  - 📅 다음 방문 일정을 확인했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 X) 시술 직후 바로 사우나를 해도 괜찮다?
     → 열과 자극은 진정에 방해돼요. 일정 기간 사우나는 피하세요.
  2. (정답 O) 외출 시 자외선 차단제를 발라야 한다?
     → 자외선은 색소·트러블의 원인! 차단제를 꼼꼼히 발라주세요.
  3. (정답 O) 충분한 보습은 시술 후 피부 진정에 도움이 된다?
     → 보습은 피부 장벽을 지켜 진정과 회복을 도와줘요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(시술 효과 유지 / 피부 트러블 예방 / 재방문율 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
