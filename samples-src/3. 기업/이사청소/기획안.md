# 기획안 — 이사 · 청소 · 방역: 서비스 이용 전후 안내 스마트 콘텐츠

> 기업 교육용 라인업. 의료 콘텐츠와 **같은 5단계 흐름**을 유지하되,
> **기업용 새 엔진·디자인**(코퍼릿 톤, 스텝 인디케이터, 적용 업종 칩, 기대 효과, 회사 연락처)으로 구성.

## 콘텐츠 개요
- **콘텐츠명**: 서비스 이용 전후 안내 스마트 콘텐츠
- **분야/업종**: 이사 · 청소 · 방역
- **적용 가능 업종**: 이사업체 · 청소업체 · 방역업체 · 입주청소 · 건물관리 · 운송·물류
- **대상**: 해당 업종 종사자/이용자 (1인용)
- **사용 환경**: 모바일 링크 및 QR코드 접속 (스마트폰/태블릿/PC)
- **예상 소요 시간**: 3~4분
- **자료 제공처**: 바이브에듀

## 화면 구성 — 총 5단계

### 1단계) 시작 화면
- 좌측: 제목·소개 + 적용 가능 업종 칩 + '학습 시작하기' 버튼
- 우측: **오늘의 학습 내용**(6항목) + **이용 안내** 패널
- 상단: 브랜드 칩(🧹 서비스 안내) + 5단계 스텝 인디케이터
- **레이아웃**: `bottombar` (하단바형) — 의료판처럼 업종별로 다른 6종 레이아웃 중 하나
- 테마색 #0EA5E9 + 포인트 #34D399

### 2단계) 핵심 학습 카드 (6장 스와이프)
  1. 📝 이용 전 준비사항 — 예약 일정과 공간 정리 등 사전 준비를 안내해요.
  2. 📦 이사 서비스 안내 — 포장·운반·배치 절차를 순서대로 익혀요.
  3. 🧽 청소 서비스 안내 — 청소 범위와 방식을 명확히 안내해요.
  4. 🧴 방역 서비스 안내 — 방역 전후 환기와 주의사항을 안내해요.
  5. ✅ 이용 후 확인 — 완료 상태를 고객과 함께 확인해요.
  6. ⚠️ 문제 발생 시 대처 — 하자·분쟁 발생 시 응대 방법을 익혀요.

### 3단계) 서비스 준비 체크리스트
  - 🗓️ 예약 일정과 범위를 확인했어요
  - 💍 귀중품을 별도로 보관했어요
  - 🧹 작업 공간을 정리했어요
  - ✅ 완료 상태를 함께 확인했어요
  - 🧾 결제·영수증을 확인했어요
- 우측: 💡 TIP 콜아웃 + '퀴즈로 넘어가기' 버튼

### 4단계) 확인 퀴즈 (OX 3문항, 문항당 +10점)
  1. (정답 O) 귀중품은 작업 전에 별도로 보관하는 것이 좋다?
     → 분실·파손을 막으려면 귀중품은 따로 보관하세요.
  2. (정답 X) 서비스 완료 후 상태는 확인하지 않아도 된다?
     → 완료 확인은 분쟁을 예방해요. 고객과 함께 꼭 확인하세요.
  3. (정답 O) 방역 작업 후에는 충분히 환기해야 한다?
     → 안전을 위해 방역 후에는 충분히 환기해주세요.

### 5단계) 완료 화면
- 좌측: "학습 완료!" + 획득 점수(정답 수 × 10점) + **기대 효과** 칩(고객 만족도 향상 / 분쟁 예방 / 재이용률 향상)
- 우측: **문의 및 안내**(전화·카카오·인스타) + **다음 행동 선택**(카드 다시 보기 / 체크리스트 다시 보기 / 처음으로)

## 기술 구성
- 공용 기업 엔진(app.ts→main.js) + 업종별 data.json (콜센터와 동일 엔진)
- 의료판과 동일 패키징(index.html / main.js / style.css / data.json / appHelper.ts)
